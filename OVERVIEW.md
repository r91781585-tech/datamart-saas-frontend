# DataMart SaaS Platform

Live demo of a mini SaaS platform built with vanilla HTML, CSS, and JavaScript.

## Quick Start

Simply open `index.html` in your browser or visit the live demo.

## Features

✅ User Authentication (Sign up/Login)
✅ 2 Sample Datasets (Startup Funding & Real Estate)
✅ Advanced Filtering System
✅ Data Preview (First 10 rows)
✅ Pay-per-row Pricing ($0.05/row)
✅ Simulated Payment Gateway
✅ Instant CSV Download
✅ Purchase History
✅ Fully Responsive Design

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Variables)
- Vanilla JavaScript (ES6+)
- LocalStorage for persistence

## Local Development

```bash
# Clone the repository
git clone https://github.com/r91781585-tech/datamart-saas-frontend.git

# Navigate to directory
cd datamart-saas-frontend

# Open in browser
open index.html

# Or use a local server
npx http-server
```

## How It Works

1. **Sign Up/Login** - Create account or login
2. **Browse Datasets** - View available datasets
3. **Apply Filters** - Filter data by multiple criteria
4. **Preview Data** - See first 10 rows
5. **Purchase** - Select rows and complete payment
6. **Download** - Get CSV file instantly
